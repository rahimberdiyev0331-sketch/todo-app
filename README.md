
# Modern Todo App

A modern and responsive Todo App built with HTML, CSS, and JavaScript.

## ✨ Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- Filter (All, Active, Completed)
- Progress tracking
- Local Storage support
- Responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
